Projeto de Desenvolvimento de Software

A OutBox Tecnologia é especializada no desenvolvimento de software e integração de sistemas, com foco em soluções SaaS e APIs.

Oferecemos soluções robustas e escaláveis que garantem a sincronização eficiente e segura de dados entre diferentes plataformas.

Nossa experiência na integração de sistemas de gestão, como o Olist Tiny ERP, nos permite transformar desafios complexos em soluções simples e eficazes, proporcionando segurança e desempenho para os nossos clientes.

Visite nosso site: outbox.inf.br

Resumo do Projeto

  • Desenvolvimento de uma API SaaS para sincronização automatizada de estoques entre diferentes empresas que utilizam o Olist Tiny ERP.
  • Sincronização em tempo real de dados de estoque, incluindo produtos, quantidades e preços, entre as empresas conectadas.
  • Criação de uma interface web para o usuário que permitirá a configuração intuitiva das integrações entre as empresas.
  • Funcionalidade para acompanhar logs detalhados das operações, permitindo monitoramento em tempo real e detecção de erros.
  • Opções de ajuste e personalização das configurações de integração através da interface gráfica.
  • Integração planejada tanto para usuários do Tiny ERP quanto do Bling, adaptando as rotinas para cada sistema conforme necessário.

Prazo de Execução e Proposta de Valores

  • Prazo de Execução: 21 dias úteis, a partir da data de assinatura do contrato.
  • Valor do Projeto: R$ 7.000,00.
  • Condições de Pagamento: 30% de entrada, seguido de 35% em 30 dias e 35% em 60 dias.

    • Observação: Estamos abertos a discutir ajustes no escopo ou nas fases do projeto para melhor se adequar às restrições de prazo e orçamento.

Suporte Técnico

  • Duração do Suporte: 30 dias de suporte técnico após o lançamento.
  • Escopo do Suporte: Inclui acompanhamento durante o Go Live e suporte para os primeiros clientes após o lançamento da página de vendas.
  • Canais de Suporte: E-mail, telefone e videoconferência.

Entregáveis do Projeto

  • Fase 1: Desenvolvimento da API

    Desenvolvimento da API de sincronização de estoque entre empresas, integrando com o Olist Tiny ERP e Bling, ajustando rotinas conforme a necessidade de cada sistema.

  • Fase 2: Implementação de MicroSaaS
    • Autenticação de usuários e recuperação de senha.
    • Notificações por e-mail para eventos importantes.
    • Integração com gateway de pagamento para assinaturas ou compras.
  • Fase 3: Go Live e Suporte Pós-Lançamento

    Lançamento do sistema com suporte técnico completo por 30 dias, incluindo acompanhamento do Go Live e suporte para os primeiros clientes.

Tecnologias Utilizadas

  • Frontend - Next.js: Framework para criação de interfaces de usuário rápidas e modernas, usado por empresas como Netflix e Twitch.
  • Backend - Node.js: Plataforma de alta performance para desenvolvimento de aplicações escaláveis, adotada por LinkedIn, Walmart, e Uber.
  • Banco de Dados - PostgreSQL: Banco de dados robusto e confiável, utilizado por Instagram, Spotify, e Airbnb para gerenciar grandes volumes de dados.
  • Infraestrutura - AWS: Serviços de nuvem escaláveis e seguros, amplamente utilizados por Netflix, Airbnb, e Slack.
  • Gateway de Pagamento - Stripe, Asaas ou Mercado Pago: Integrações seguras e eficientes, usadas por Amazon, Shopify, e Mercado Livre.

Vantagens das Tecnologias Utilizadas

  • Escalabilidade: As tecnologias utilizadas, como Next.js e Node.js, são projetadas para escalar facilmente conforme o crescimento do negócio.
  • Desempenho: Aplicações desenvolvidas com essas tecnologias oferecem melhor desempenho, especialmente em operações complexas e de alto volume.
  • Flexibilidade: A utilização de tecnologias como PostgreSQL e AWS permite customizações avançadas e integração com outros sistemas de maneira mais eficiente.
  • Segurança: O uso de plataformas robustas como AWS garante um nível de segurança elevado, com controle total sobre os dados e a infraestrutura.
  • Controle Total: Tecnologias como Node.js e Next.js permitem controle completo sobre o código e a arquitetura, proporcionando a possibilidade de ajustes finos que são impossíveis em soluções no-code/low-code.
